# Environment Variables Examples

This directory contains progressive examples teaching environment variables in Python.

**Course materials**: https://finm-32900.github.io/Week2/env_files.html

## Examples

| Directory | Topic |
|-----------|-------|
| `01_cli_basics` | Set env vars on command line, access with `echo` and Python |
| `02_dotenv_file` | Load variables from `.env` files using `python-dotenv` |
| `03_decouple` | Use `python-decouple` with defaults and type casting |
| `04_path_exploration` | Understand the PATH variable |
| `05_wrds_credentials` | Practical example: storing WRDS credentials |

Work through them in order.
